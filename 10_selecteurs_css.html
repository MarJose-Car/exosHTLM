Les sélecteurs css

  • *:selectionne TOUS les éléments de la page HTML
  • .ma-jolie-class: selectionne tous les éléments ayant le nom de class "ma-jolie-class"
  • #mon-unique-id:sélectionne tous les éléments ayant le nom d'ID "mon-unique-id"
  • div.aside.article:selectionne toutes les balises <div>, <aside>, <article>
  • nav a:

Propriétes CSS utilisées dans cette page

Décrivez leurs usage et leurs valeurs possible

  1. Propriétés pour le text
    • test-align : alignement du text, à gauche, à droit, à droite ou au centre :Syntax: start | end | left | right | center | justify | match-parent
    • font-family :spécifie les caractères à utiliser en priorité: Syntax: family-name
    • font-size :indique l'hauter des caractères: Syntax: <absolute-size> relative-size>length-percentage>
    • text-decoration :pour "décorer" le texte en ajoutant une ligne en dessous
    • color:pour la couleur du text :couleur de la typo, de la font du texte
    • line-height espace entre les lignes du texte ou des boites:Syntax: normal <number>length>percentage>percentage>

  2. Propriétes pour les éléments
    • background: pour la couleur de fond d'un élément
    • float:specifie dans quel direction une boite est positionnée :Syntax: left | right | none | inline-start | inline-end
    • height :spécifie l'hauteur de l'élément: Syntax: <viewport-length>{1,2}
    • width :largeur de l'élément: tous les deux par rapport à la taille de la fenêtre si on indique une valeur proportionnelle en % ou en em
    • left :position à partir de la gauche
    • margin :marge extérieure d'un élément
    • padding:marge intérieur d'un élément
    • position :position d'un élément d'un la page
    • top :position par rappart au haut de la fenêtre
    • transiction : temps de transiction lors d'une animation
