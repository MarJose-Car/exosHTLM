  • font-weight: La propriété CSS font-weight permet de spécifier le type de gras à utiliser pour l'affichage de la fonte.Exemple d'écriture CSS de font-weight font-weight : normal; font-weight : bold; font-weight : bolder; font-weight : 800; La propriété de feuille de style font-weight CSS pour mettre en gras peut p...:specifie l'hauteur des caractères et leurs expessure: Syntax: <font-weight-absolute>{1,2}
  • letter-spacing: :La propriété letter-spacing définit l'interlettre utilisée pour les caractères qui composent le texte./* Valeurs de longueur */ /* Type <length> */ letter-spacing: 0.3em; letter-spacing: 3px; letter-spacing: .3px; /* Valeur avec mot-clé */ letter-spacing: normal; /* Valeurs globales */ letter-spacing: inherit; letter-spacing: initial; letter-spacing: unset; specifie le minimum,le maximum et l'optimil space entre les lettres: Syntax: normal Valeurs normal L'interlettre est dictée par la police utilisée. À la différence de 0, cette valeur permet à l'agent utilisateur de modifier l'espace entre les caractères afin de le justifier. <length> Cette valeur indique l'espace ajouté à l'espace par défaut. Les valeurs peuvent être négatives mais les limites imposées dépendent de l'implémentation du moteur. Les agents utilisateurs ne pourront plus augmenter ou réduire l'interlettre afin de justifier le texte. Pour les différentes unités possibles, se référer à la page . | <length>length>

  • opacityopacité d'un élément text, where 1 is opaque et 0 completement transparent:Syntax: <alpha-value>alpha-value>
  • border-stylepropriété pour la bordure, taille, hauteur, marge extérieur, marge intérieur, style et couleur;Syntax: <line-width>line-style>color>
  • box-shadowLa propriété CSS box-shadow ajoute des ombres à la boîte d'un élément via une liste d'ombres séparées par des virgules
  • liste-style-image La propriété list-style-image définit l'image utilisée comme puce devant les éléments de listes.
  • borderborder est une propriété raccourcie qui permet de définir les propriétés liées à la bordure.
  • border-radiusborder-radius permet de définir des coins arrondis pour la bordure d'un élément.
  • border-widthborder-width est une propriété raccourcie qui définit la largeur de la bordure d'un élément.
  • font-vatiant La propriété font-variant-caps permet de contrôler les glyphes utilisés pour représenter les lettres en capitales.
  • background-colorLa propriété background-color permet de définir la couleur utilisée pour l'arrière-plan d'un élément
  • !importantLa règle CSS !important permet de spécifier que la propriété est la plus importante dans votre feuille CSS en lui donnant un poids plus important.La règle de feuille de style !important CSS est généralement utilisée dans le cas où une propriété de feuille de style CSS est déclarée plusieurs fois avec des ciblages différents.La règle CSS !important est applicable sur toutes les feuilles de style et ne concerne que la propriété CSS sur laquelle on a appliqué la règle. La règle de feuille de style !important CSS n'a pas d'équivalence en HTML ou XHTML. La règle CSS !important est à éviter si possible.
